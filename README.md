## 🚀 快速开始

### 1. 了解项目
Qt节点编辑器是一个基于Qt框架的可视化节点编程工具，支持：
- ✅ 可视化节点编辑和连接
- ✅ 多种节点类型（输入、滤波器、变换、输出）
- ✅ 实时连接线更新和同步
- ✅ 代码生成和导出功能
- ✅ JSON格式数据序列化

### 2. 运行程序
```bash
# Windows
CodeGenerator.exe

# Linux/Mac
./CodeGenerator
```

### 3. 创建第一个流程图
1. 启动程序
2. 点击"节点" → "添加输入节点"
3. 点击"节点" → "添加滤波器节点"
4. 点击"节点" → "添加输出节点"
5. 连接节点：点击输出端口 → 拖拽到输入端口
6. 保存文件

### 4. 生成代码
1. 点击"工具" → "生成C++代码"
2. 选择输出位置
3. 编译运行生成的代码

## 🎯 核心功能特性

### 节点管理
- **拖拽移动**: 支持节点自由拖拽和精确定位
- **端口检测**: 智能端口检测和高亮显示
- **批量操作**: 支持多选、复制、粘贴操作
- **属性编辑**: 双击节点编辑参数和属性

### 连接系统
- **智能连接**: 自动端口检测和对齐
- **实时更新**: 连接线跟随节点移动实时更新
- **贝塞尔曲线**: 平滑的连接线视觉效果
- **状态管理**: 完整的连接状态跟踪

### 代码生成
- **多语言支持**: 生成C++、Python、JavaScript代码
- **依赖分析**: 自动分析节点依赖关系
- **执行排序**: 基于拓扑排序的正确执行顺序
- **配置生成**: 生成部署配置文件

### 调试支持
- **详细日志**: 分层调试输出系统
- **状态监控**: 实时连接状态监控
- **错误检测**: 自动检测循环依赖等错误
- **性能分析**: 流程分析和优化建议

## 🏗️ 技术架构

### 组件结构
```
Qt节点编辑器
├── UI层
│   ├── MainWindow - 主窗口和界面管理
│   ├── NodeView - 视图显示和交互
│   └── 菜单和工具栏
├── 场景管理层
│   └── NodeScene - 节点和连接管理
├── 数据模型层
│   ├── Node - 节点数据模型
│   └── Connection - 连接数据模型
└── 代码生成层
    └── CodeGenerator - 代码生成和分析
```

### 设计模式
- **MVC架构**: 分离数据、视图和控制逻辑
- **观察者模式**: 节点变化通知连接线更新
- **策略模式**: 不同节点类型的处理策略
- **工厂模式**: 节点和连接的创建管理

## 🔧 安装和部署

### 系统要求
- **操作系统**: Windows 10+, Linux, macOS
- **Qt版本**: 6.0+
- **编译器**: MSVC 2019+, GCC 7+, Clang 6+
- **内存**: 最小512MB，推荐2GB+
- **存储**: 最小100MB可用空间

### 编译环境
```bash
# 使用qmake
qmake CodeGenerator.pro
make

# 或使用CMake
cmake .
make
```

### 部署包
- **Windows**: CodeGenerator_v1.0.0_Windows.zip
- **Linux**: CodeGenerator_v1.0.0_Linux.tar.gz
- **macOS**: CodeGenerator_v1.0.0_macOS.dmg
